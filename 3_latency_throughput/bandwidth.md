# Bandwidth

**Bandwidth** is the maximum amount of data that can be transmitted over a network in a given amount of time.

---

## Simple Meaning

> Bandwidth = maximum capacity of a network

It tells you how much data _can_ be sent, not how much is actually sent.

---

## Example

If a network has:

- Bandwidth = 100 Mbps

It means up to **100 megabits per second** can be transmitted under ideal conditions.

---

## Analogy

Think of a highway:

- Bandwidth = number of lanes
- More lanes → more cars can move at the same time

So:

- Wide road → high bandwidth
- Narrow road → low bandwidth

---

## Units of Bandwidth

- bps (bits per second)
- Kbps
- Mbps
- Gbps

---

## Key Point

> Bandwidth is the **maximum possible data transfer rate**, not the actual speed.
